Viewer 1.0 (PDFViewerDB)

概要
デジタルコミックを読み込んで閲覧するのに使うソフトウェア

取り扱い種別 フリーソフト
これから寄付の受付もしてみようかと思っています。購入の条件ではありません。

インストール・アンインストール
解凍してできた実行ファイルを起動させれば使えます。
現在レジストリに書き込む仕様となりましたがアンインストールで削除に対応しておりません。
クリーンなアンインストールはあきらめてください。申し訳ありません。


使用に際して
画像を読み込みます。メインメニューから行えます。
登録されたデータを選択すると閲覧が開始します。

見開き機能、画像の見開きペアがずれていた時のために表紙あり設定がついています。
感覚的ではありますが左開きか右開きか選択できます。
自動捲り機能もあります。

ページ画像クリックでページ送りページ戻しになります。下部のトラックバーでも調整できます。
そのトラックバーの調整には十字キーも使えます。
